For APID, this is not compulsory, but it is highly recommended.
A lightweight template repository for coding and LaTeX exercises, plus quick-reference guides for common developer workflows.

When opening in CODESPACES, the repository will automatically set up a development environment with all necessary tools and dependencies installed.
It will take about 10 minutes for the first time.
However, once set up, subsequent openings will be much faster.
